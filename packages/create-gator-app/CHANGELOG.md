All notable changes to this project will be documented in this file.

### Changed

- Rename ERC-7715 template to Advanced Permissions template
- Update templates to use `@metamask/smart-accounts-kit`

- Fix Farcaster template dependency issue

## [0.1.3]

### Changed

- Update ERC-7715 template to latest v0.13

### Added

## [0.1.1]

### Added

- Adds MetaMask Smart Accounts template

### Changed

- Update templates to latest v0.12

### Fixed

- Fix dependabot alerts

### Uncategorized

- Initial release

## Added

- Starter templates for the Delegation Toolkit with Next.js and Vite (React)
- ERC-7715 starter template for Next.js
- Support for feature flags
